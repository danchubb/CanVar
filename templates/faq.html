Frequently Asked Questions

General

  • {{name}} Looks a lot like ExAC...

    {{name}} was created based upon the ExAC framework provided on github and as such it shares much of the design. We give full credit to the MacArthur team for providing this invaluable resource. However, while ExAC is incredibly useful for looking up population variant frequencies, it does not provide phenotype specific frequencies. {{name}} has been created to specifically provide study level frequency data for germline exome studies in cancer.

  • What cancer studies are included in {{name}} ?

    Currently {{name}} contains variant level data for 1,006 NSCCG familial early onset colorectal cancer patients sequenced at the Institute of cancer research. It is anticipated that further germline cancer studies will be added in the near future.

  • How should I cite discoveries made using {{name}} data?

    We anticipate the publication of a {{name}} paper describing this data set in mid 2016. If the NSCCG CRC data is used, please cite : Chubb et-al. The specific NSCCG calls made in CanVar were jointly alongside a larger set of exomes sequenced at the ICR. Calls may therefore be different from those presented in the original publication.

  • Can I get access to individual-level genotype data from {{name}}

    Indivudual level data on the 1,006 NSCCG CRC cases have been deposited at the European Genome-phenome Archive, which is hosted by the European Bioinformatics Institute, accession number: EGAS00001001666

Technical details

  • What genome build is the {{name}} data based on?

    All data are based on GRCh37/hg19

  • What version of dbSNP was used to annotate variants?

    dbSNP 135. The browser supports RSIDs up to dbSNP 141, but these are not included in the VCF.

  • What version of Gencode was used to annotate variants?

    Version 19 (annotated with VEP version 77).

  • How was CanVar built?

    The adapted ExAC framework and parsers can be found on github
